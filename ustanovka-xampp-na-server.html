Установка XAMPP на сервер

Подготовка и загрузка

Сначала необходимо обновить список пакетов и перейти в директорию для загрузки (например, /tmp).

sudo apt update
cd /tmp

Теперь нужно скачать установщик. Последнюю версию можно найти на официальном сайте Apache Friends.

Скопировать ссылку на установщик для Linux (файл с расширением .run), использовать wget для загрузки. Например, для версии с PHP 8.2.12

wget https://sourceforge.net/projects/xampp/files/XAMPP%20Linux/8.2.12/xampp-linux-x64-8.2.12-0-installer.run

Настройка прав и запуск установки

Сделать скачанный файл исполняемым и запустить его с правами суперпользователя.

sudo chmod +x xampp-linux-x64-8.2.12-0-installer.run
sudo ./xampp-linux-x64-8.2.12-0-installer.run

Управление службами XAMPP

Запустить все службы (Apache, MySQL, FTP):

sudo /opt/lampp/lampp start

Остановить все службы:

sudo /opt/lampp/lampp stop

Проверить статус:

sudo /opt/lampp/lampp status

Перезапустить (после изменения конфигурации):

sudo /opt/lampp/lampp restart

Проверка работы

Открыть в браузере адрес сервера: http://ip-вашего-сервера. Вы должны увидеть панель приветствия XAMPP.

phpMyAdmin будет доступен по адресу: http://ip-вашего-сервера/phpmyadmin.
